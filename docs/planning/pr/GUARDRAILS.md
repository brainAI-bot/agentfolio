# AgentFolio PR Guardrails

## Mode

Organic only. No paid spend, cold DMs, cold email, public-launch claims, partnership claims, legal handling, complaint handling, money handling, credential handling, Solana writes, keypair handling, or unsupported numbers.

## Rate Caps

| Channel family | Daily cap | Notes |
| --- | ---: | --- |
| X | 5 replies, 3 reposts | Repost only board-flagged AgentFolio/SATP `@brainAI_` posts. |
| GitHub | 8 comments | Only relevant issues, PRs, or discussions. |
| Moltbook | 5 comments | Founder/launch/agent-builder relevance required. |

## Capability

| Channel family | Topic search | Own-account mention read | Reply post | Credential binding on braingrowth |
| --- | --- | --- | --- | --- |
| X | Pending runtime receipt | Pending runtime receipt | Blocked until Qc receipt | Present/not-present pending runtime readback |
| GitHub | Known receipt per HQ task | Known receipt per HQ task | Known receipt per HQ task | Present through GitHub CLI auth; repo clone succeeded |
| Moltbook | Known receipt per HQ task | Known receipt per HQ task | Known receipt per HQ task | Present/not-present pending runtime readback |

## Escalation

Escalate to Hani with `needs_hani=1` only when irreducible: paid spend, account/credential/permission changes, or content with legal, financial, or safety risk.
