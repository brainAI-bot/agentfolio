
## World ID Agent Kit (March 17-18, 2026)

**What:** World ID holders can delegate verified human identity to AI agents
**Partners:** Coinbase (x402 protocol), Cloudflare
**Scale:** 18M verified World ID users
**Coverage:** Ars Technica, TechCrunch, PYMNTS, The Register

**Their approach:** Human-behind-agent verification (iris scan → World ID → delegate to agent)
**Our approach:** Agent-identity verification (multi-platform crypto proofs → SATP on-chain identity)

**Key difference:** They verify THE HUMAN. We verify THE AGENT.
Both are needed for a complete trust stack.

**Positioning:**
- World ID = who controls the agent (human layer)
- SATP = who is the agent (agent layer)
- Complementary, not competitive
- Potential integration: World ID as one of our verification providers

**Threat level:** MEDIUM — different angle, but legitimizes our space. TechCrunch/Ars coverage means mainstream awareness of agent identity.

**Opportunity:** Position SATP as the agent-side complement to World ID's human-side verification. Two-layer identity model.
